## 🌐 About the Project

<a name="description"></a>

### 📖 Description

_Expense Tracker_ is a personal finance tool that allows users to manage their daily expenses. Users can add, edit, and delete expenses with descriptions, amounts, and dates. Expenses persist in the browser using localStorage and can be sorted by date.

Built as a TypeScript practice project to sharpen React and TypeScript fundamentals before returning to full stack development.

<a name="getting-started"></a>

## 🏁 Getting Started

<a name="prerequisites"></a>

### 📋 Prerequisites

⭐ You must have Node.js and npm installed on your computer.

<a name="setup-and-use"></a>

### ⚙️ Setup and Use

#### Cloning

1. Navigate to the [Expense Tracker repository here](https://github.com/onurkaymak/expense-tracker-ts).
2. Click 'Clone or download' to reveal the HTTPS url ending with .git.
3. Open your Terminal or GitBash and navigate to your desktop: `cd Desktop`
4. Clone the repository: `$ git clone https://github.com/onurkaymak/expense-tracker-ts.git`
5. Run `cd expense-tracker-ts` to enter the project directory.
6. Run `code .` to open in VS Code.

#### Installation

1. Run `npm install` to install dependencies.
2. Run `npm run dev` to start the development server.
3. Open `http://localhost:5173` in your browser.

#### Or simply visit the live app:

🌐 [https://expense-tracker-ts-orpin.vercel.app/](https://expense-tracker-ts-orpin.vercel.app/)
